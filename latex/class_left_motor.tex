\doxysubsection{Detailed Description}
Two derived classes, one for each motor. With the way Pololu controls the speeds, this avoids ugly lookup tables (though it\textquotesingle{}s not beautiful itself). Friended to \mbox{\hyperlink{class_chassis}{Chassis}} so that \mbox{\hyperlink{class_left_motor_a2c05a99b5d81aa6584773dded947bfe1}{set\+Effort()}} can\textquotesingle{}t be called from main(). Use \mbox{\hyperlink{class_chassis_aa331bdac0a5b09766c9e60cfe6ddb9e5}{Chassis\+::set\+Motor\+Efforts()}} to set efforts, as that will adjust the control mode properly.

Because the Pololu library is based on the Fast\+GPIO library, we don\textquotesingle{}t/can\textquotesingle{}t use analog\+Write. Instead, we set the duty cycle directly at the register level.

We also have to have separate classes for the left and right motors to avoid the complex mapping of speeds to registers.
